5  Likelihood

The likelihood function is the joint density (or probability mass) function of the data:

\[ f(\underline{x} \mid \theta), \quad \underline{x} = (x_1, \dots, x_n). \]

5.1 From Joint Density to Likelihood

Suppose we observe \(n\) independent and identically distributed (i.i.d.) samples,

\[ X_1, \dots, X_n \overset{\text{iid}}{\sim} f_X(x \mid \theta). \]

Then the joint density is given by multiplying the individual densities:

\[ f(\underline{x} \mid \theta) = \prod_{i=1}^n f_X(x_i \mid \theta). \]

TipRemark (Why products?)

Remark 5.1 (Why products?).

  • Independence means that joint probabilities factorise into products:

    \[ P(A \cap B) = P(A)\,P(B). \]

  • By the same rule, the joint density of independent random variables is the product of their individual densities.

  • This is often the first place where students forget the independence assumption! Without it, you cannot just multiply.

5.2 Likelihood Function

NoteDefinition

Definition 5.1 Likelihood Function. For fixed observed data \(\underline{x}\), the likelihood function is defined as

\[ L(\theta ; \underline{x}) \coloneqq f(\,\underline{x} \mid \theta). \]

Here we treat \(\theta\) as the variable, and the data \(\underline{x}\) as fixed.

TipLikelihood vs. Probability:
  • PDF/PMF: function of \(\underline{x}\) (data random, parameter fixed).
  • Likelihood: function of \(\theta\) (data fixed, parameter variable).

5.2.2 Computing Likelihood Functions

This is a very important part of the course. ?sec-binom-text

Example 5.1 (Bernoulli Likelihood) Suppose we have \(n\) i.i.d. data distributed according to the Bernoulli distribution 1.1:

\[ Y_i \mid p \sim \textrm{Bernoulli}(p). \] Derive the likelihood function.

Example 5.2 (Bernoulli Likelihood) Suppose \(n=5\) trials give data \(\underline{x} = (1,0,1,1,0)\). Then \(\sum\_i x\_i = 3\). \[ L(p ; \underline{x}) = p^3 (1-p)^2. \]

  • If \(p=0.5\), \(L(0.5) = 0.5^3 \cdot 0.5^2 = 0.03125\).
  • If \(p=0.7\), \(L(0.7) = 0.7^3 \cdot 0.3^2 \approx 0.0309\).
  • If \(p=0.8\), \(L(0.8) = 0.8^3 \cdot 0.2^2 = 0.0205\).

So the data are slightly more likely under \(p=0.5\) than \(p=0.8\).

5.3 Maximum Likelihood Estimation

NoteDefinition (Maximum Likelihood Estimator (MLE))

Definition 5.2 (Maximum Likelihood Estimator (MLE)) The MLE is the parameter value \(\hat{\theta}\) that maximises the likelihood function: \[ \hat{\theta}_{\text{MLE}} = \arg \max_{\theta \in \Theta} L(\theta \mid \underline{x}). \]

Because logs are monotone, it is usually easier to maximise the log-likelihood:

\[ \ell(\theta ; \underline{x}) = \log L(\theta ; \underline{x}). \]

5.3.1 Computing MLEs

Example 5.3 (Bernoulli MLE) From Example 5.1, we have

\[ L(\theta ; x) \]

Compute the MLE of \(p\).

TipSolution

Solution 5.1. From the provided likelihood, we first compute the log-likelihood:

\[ \ell(p \mid \underline{x}) = \Big(\sum_i x_i \Big)\log p + \Big(n-\sum_i x_i\Big)\log(1-p). \] Differentiate and solve: \[ \frac{\partial}{\partial p}\ell(p \mid \underline{x}) = \frac{\sum_i x_i}{p} - \frac{n - \sum_i x_i}{1-p} = 0. \] This gives \[ \hat{p}_{\text{MLE}} = \frac{1}{n}\sum_{i=1}^n x_i, \]

the sample proportion of successes.

Example 5.4 (Binomial MLE) Suppose we have a single Binomial 1.2 observation \[ X \mid \theta \sim \textrm{Bin}(n, \theta). \] Derive its likelihood function.

TipSolution

Solution 5.2. Since we only have one observation, the likelihood function is just the distribution of the observation: \[ L(\theta ; x) = f(x \mid \theta) = \binom{n}{x} \theta^x(1-\theta)^{n-x}. \] The log-likelihood is thus \[ \ell(\theta; x) = \log f(x \mid \theta) = \log\left(\binom{n}{x}\right) + x\log\theta + (n-x)\log(1-\theta). \] Differentiating1, we obtain \[ \frac{\partial}{\partial \theta} \]

Example 5.5 (Binomial MLE) Suppose we have \(n\) independent Binomial data: \[ X_i \mid \theta \sim \textrm{Bin}(n_i, \theta). \] Note here that each Binomial data has a different number of Bernoulli trials \(n_i\), which are known.

Derive its likelihood function.

TipSolution

Solution 5.3. The likelihood function is the joint distribution of the \(n\) observations \[ f(\underline{x} \mid \theta) = \prod_{i=1}^n \binom{n_i}{x_i} \theta^{x_i}(1-\theta)^{n_i - x_i}. \]

Important✨ Big picture:
  • Likelihood tells us how plausible each parameter value is, given the data.
  • The MLE chooses the parameter value that makes the observed data most likely.
  • Later, in Bayesian inference, we will combine the likelihood with a prior distribution on \(\theta\) to obtain a posterior distribution.
